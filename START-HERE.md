## 🎯 立即体验

### 1. 测试撤销/重做功能
```bash
npm run dev
# 打开 http://localhost:9999/
```

在编辑器中：
1. 输入一些文字
2. 按 `Ctrl+Z` 撤销 → 文字消失 ✅
3. 按 `Ctrl+Y` 重做 → 文字恢复 ✅

A: 执行 `npm run build` 验证打包，然后继续测试剩余功能。

**Q: 如何使用撤销/重做？**  
A: 按 `Ctrl+Z` 撤销，`Ctrl+Y` 重做，或调用 `editor.commands.undo()/redo()`

**Q: 日志如何使用？**  
A: `import { logger } from '@utils/logger'` 然后使用 `logger.debug()` 等方法。
